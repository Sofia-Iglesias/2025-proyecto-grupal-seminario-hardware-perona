# **`Lámpara ambientadora`**

`Tecnología de los Sistemas de Información - Seminario Avanzado`  
`Ciclo Lectivo 2025`

## **`Integrante`**

- `Máximo Perona`  

## **`Objetivo`**

`El objetivo es crear una lampara LED que contenga otros actuadores que generen una ambientación distinta en la habitación dependiendo del clima que es detectado por sensores propios del sistema. Esto crea una ambientación especifica para cada momento`

## **`Descripción del Proyecto`**

Este proyecto busca crear un dispositivo que se adapte a tu ambiente, no distorcionando la luz ambiental sino que acompañandola. Para eso, busca ver las condiciones ambientales en cada
momento y de ahí cambiar el patrón de colores.

## **`Componentes`**

-Anemómetro

-Sensor lluvia

-Fotoresistor

-Tira NeoPixel de 8 pixeles

-Boton para prender y apagar

## **`Requisitos`**

Utiliza la libreria <Adafruit_Neopixel.h>

- Simular todo el proyecto en tinkercad

### Etapa 2:
- Debuguear la simulación
