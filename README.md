## **`Objetivo`**

`El objetivo es crear una lampara LED que contenga otros actuadores que generen una ambientación distinta en la habitación dependiendo del clima que es detectado por sensores propios del sistema. Esto crea una ambientación especifica para cada momento`

## **`Descripción del Proyecto`**

`Explicar en qué consiste y qué se busca lograr. Completar una vez definido el proyecto.`

## **`Componentes`**

-Sensor lluvia

-Fotoresistor

-Tira NeoPixel de 8 pixeles

-Boton para prender y apagar

-Una carcaza diseñada en 3D que contenga los actuadores del sistema de manera estetica y compacta para poder ser instalada en cualquier parte. debe de tener un "compartimiento" que contenga la tira LED con un difusor de luz para que sea estetico

Utiliza la libreria <Adafruit_Neopixel.h>
